# McMaster Computer Society Web Hub

A collaborative web platform developed for the McMaster Computer Society to centralize event management, user interaction, and organization resources.  
The project was built by a team of first-year Computer Science students using front-end and back-end web technologies.  
The website includes responsive pages, event registration functionality, user account features, and database integration to support student engagement and organization activities.  
This project strengthened my experience working in a collaborative development environment and contributing to a larger-scale web application.

## Technologies Used
- PHP
- JavaScript
- HTML
- CSS
- MySQL

## Features
- User authentication and account management
- Event registration and participation system
- Responsive layouts for desktop and mobile devices
- Interactive navigation and user interface components
- Database integration for storing and retrieving user/event data

## How to Run
1. Install XAMPP or another local PHP server environment
2. Place the project folder inside the `htdocs` directory
3. Start Apache and MySQL services
4. Import the provided database into phpMyAdmin
5. Open the project in your browser:
   http://localhost/css_hub/css_hub/public/index.php
6. Login with macid and password or create an account
